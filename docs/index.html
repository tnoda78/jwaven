jwaven

CLI tool for getting "J-WAVE on air" song list.

Download

Visit release page

Repository

https://github.com/tnoda78/jwaven/

Usage


        jwaven [options]
        --now -n
          今かかっている曲
        --latest -l
          現在時刻の前後60分の楽曲
        --yesterday -y
          昨日の時刻の前後60分の楽曲
        --date -d {yyyy-MM-dd hh:mm}
          指定日時の前後60分の楽曲
        --format -f {standard,searchgoogle,json,csv}
          指定フォーマットで出力をする

Example


        jwaven --now
        

        jwaven --latest
        

        jwaven --latest --format json | jq .
        

        jwaven --latest --format csv > jwaven.csv
        cat jwaven.csv
