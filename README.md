![image](https://tnoda78.github.io/jwaven/01.gif)

```
jwaven [options]
    --now -n
      今かかっている曲
    --latest -l
      現在時刻の前後60分の楽曲
    --yesterday -y
      昨日の時刻の前後60分の楽曲
    --date -d {yyyy-MM-dd hh:mm}
      指定日時の前後60分の楽曲
    --format -f {standard,searchgoogle,searchgpm,searchyoutube,json,csv}
      指定フォーマットで出力をする
```
